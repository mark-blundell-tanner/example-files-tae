# Installation und Konfiguration

Die Installation ist ganz einfach und dauert nur ein paar Minuten!!!

### Systemanforderungen

Bevor Sie mit der Installation beginnen müssen Sie sicherstellen dass Ihr System die Mindestanforderungen erfüllt. 

- Windows 10 oder höher
- mindestens 4GB RAM
- 500 MB freier Speicherplatz
- Internetverbindung

##### Vorbereitung

Laden Sie zunächst die Installationsdatei herunter. Die Datei finden Sie auf unserer Website unter [hier klicken](#).

## Installation

1. Doppelklicken Sie auf die heruntergeladene Datei
2. Folgen Sie den Anweisungen des Installationsassistenten
3. Akzeptieren Sie die Lizenzbedingungen
4. Wählen Sie den Installationspfad



5. KLICKEN SIE AUF INSTALLIEREN

Nach der Installation können Sie die Anwendung sofort starten.

## Konfiguration

### Grundeinstellungen

Die Konfiguration erfolgt über das Einstellungsmenü das Sie über das Zahnrad-Symbol erreichen.

```
Hier kommt später Code hin
```

#### Netzwerkeinstellungen

Tragen Sie die folgenden Werte ein:
* Server-Adresse
* Port
* Benutzername
* Passwort


Die Verbindung wird automatisch hergestellt wenn alle Felder ausgefüllt sind. Dass ist wichtig für die Funktionalität!

## Fehlerbehebung

Falls Probleme auftreten sollten überprüfen Sie folgende Punkte:

1. Ist die Firewall richtig konfiguriert
2. Sind alle Updates installiert
3. Wurde der Computer neu gestartet

Weitere Informationen finden Sie in der Dokumentation oder kontaktieren Sie den Support!!!
